Le projet en deux phrases

Ce dashboard explore comment la deviation du spectre empirique d'une matrice de correlation a l'universalite GUE (equivalente a la distribution des zeros de la fonction zeta de Riemann) peut etre utilisee comme feature predictive pour la volatilite multi-asset. Six modeles d'apprentissage (Ridge, Random Forest, Gradient Boosting, MLP-Adam, Monte Carlo k-NN) sont compares de facon transparente sur un test set hors echantillon de 193 fenetres.

Cadre mathematique

La conjecture de Montgomery-Odlyzko etablit que les espacements normalises des zeros non-triviaux de la fonction zeta de Riemann suivent la distribution attendue pour les eigenvalues d'une matrice hermitienne aleatoire (ensemble GUE), donnee par la surmise de Wigner :

p_GUE(s) = (32 / pi2) · s2 · exp( -(4/pi) · s2 )

Pour une matrice de correlation empirique, toute deviation a cette distribution signale une structure non-aleatoire (concentration de variance, contagion). Nous mesurons cette deviation par une distance de Kolmogorov-Smirnov, et construisons le zeta-score = log(D_KS_Poisson / D_KS_GUE).
